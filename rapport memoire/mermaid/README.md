# Diagrammes Mermaid — Mémoire BlockTask (Chapitre 3)

## Visualiser en ligne (recommandé)

1. Ouvrez **[Mermaid Live Editor](https://mermaid.live)**.
2. Ouvrez un fichier `.mmd` de ce dossier.
3. Copiez tout le contenu → collez dans l’éditeur.
4. Le diagramme s’affiche à droite.
5. **Exporter** : bouton *Actions* → PNG / SVG / PDF pour Word.

## Conseils pour Word

- Export **PNG** en 1920px ou plus pour une bonne lisibilité.
- Légende sous chaque figure : *Figure 3.X — Titre du diagramme*.
- Référencez les figures dans le texte : *« comme illustré à la Figure 3.7 »*.
